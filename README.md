# Login flow with NestJS, Express and Mongo



## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```



## Queries 

1. Signup/register account
3. Login with password
4. Change password
5. View details of acount
